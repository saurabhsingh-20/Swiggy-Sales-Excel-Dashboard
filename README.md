# 🍔 Swiggy Sales Analysis Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive **Microsoft Excel Dashboard** built using a real-world Swiggy sales dataset. The objective was to analyze food delivery sales, customer ratings, order patterns, and regional performance to generate actionable business insights.

The dashboard enables users to monitor key business metrics and explore sales trends using interactive filters and visualizations.

---

## 🎯 Business Problem

Food delivery companies generate large volumes of transactional data every day. Without effective analysis, it becomes difficult to identify:

- Sales trends over time
- Best-performing cities and states
- Customer ordering behavior
- Performance across different quarters
- Revenue contribution by food category

This dashboard provides a single view to monitor business performance and support data-driven decision making.

# 🛠 Data Preparation

Before building the dashboard, the dataset was cleaned and enhanced by:

- Removing inconsistencies
- Creating **Week** column for weekly analysis
- Creating **Quarter** column for quarterly reporting
- Adding **Veg / Non-Veg** category for food type comparison
- Creating Pivot Tables for dashboard visualization
